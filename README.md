# 🦯 Smart Blind Stick

## 📌 Description
The Smart Blind Stick is an assistive device designed for visually impaired people. It helps users detect obstacles and navigate safely using sensors and alert systems.

This project is developed using Arduino Nano and C++ programming.

## 🔧 Components Used
- Arduino Nano  
- Ultrasonic Sensor  
- IR Sensor  
- Piezo Buzzer  
- Vibration Motor  
- LED with Resistor  
- 3.7V Li-ion Battery  
- Jumper Wires  
- On/Off Switch  

---

## 💡 Features
- Detects obstacles using ultrasonic sensor  
- IR sensor detects nearby objects and ground changes (like steps or holes)  
- Provides alert using buzzer sound  
- Provides vibration alert using motor  
- LED indication for visual feedback  
- Low-cost and easy to use  

---

## ⚙️ Working Principle
The ultrasonic sensor continuously checks for obstacles in front of the user by measuring distance.

The IR sensor detects nearby objects or ground changes such as steps or holes.

When an obstacle is detected:
- The buzzer produces sound  
- The vibration motor vibrates  
- The LED turns ON  

This helps the user understand their surroundings and move safely.
